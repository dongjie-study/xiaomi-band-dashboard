一 话术违规识别 · 单选题（每题3分，共30分）
👉 每题给出一种主播口播场景，判断属于什么违规类型
1主播在镜头前说："买我们的手环享受全国联保，一年官方质保，线上线下都可以维修。"但详情页仅标注"官方保修"。这属于什么违规？
✅ 正确答案：B — 售后服务不符
"全国联保""一年质保""线上线下都可以维修"均属于售后承诺。若商品详情页未明确标注这些承诺，口播即构成违规。耳机直播间曾因此类话术被冻结佣金14,650元/30天+关闭商品分享3天。 📎 案例：违规单号 7642750111487148331、7642986916338123062、7543509078755770086
🚫 高危话术：全国联保 / 一年质保 / 线上线下都能修 / 无忧质保 / 品质质保 / 终生售后
✅ 安全替代：详情页有写什么就说什么 / "具体售后政策可以在详情页查看" / 引导看商品描述
2主播说："这个REDMI Watch支持五十米深度防水，游泳洗澡潜水都没问题！"但产品实际仅支持IP68生活防水。这属于什么违规？
✅ 正确答案：C — 效果虚假/夸大宣传
5月6日小米官方手表直播间真实案例：口播"五十米深度防水"被判定为效果虚假/夸大宣传。处罚：下架商品+冻结佣金5%+中断直播+关闭商品分享1天。 📎 案例：违规单号 7536494293812658473
🚫 高危话术：五十米防水（实际IP68） / 能测血糖血压（实际不能） / 戴着手表潜泳没问题 / 摔不坏 / 刀刮不花
✅ 安全替代：严格按产品说明书上的参数描述 / IP68生活防水 / 日常洗手淋雨没问题
3主播为活跃气氛说："来，点赞到5000！到5000我抽3个人，一人送一个手环！扣1的宝宝优先！"（未使用平台福袋工具）这属于什么违规？
✅ 正确答案：D — 双重违规
"点赞到5000就抽"=诱导互动（以利益诱导用户行为）；"手动抽"=未使用福袋工具。两个问题叠加，6月已有2次真实违规记录。 📎 案例：违规单号 7543943557177331746、7656231531917039318、76530149197860
🚫 高危话术：扣1抽奖 / 点赞到XX就送 / 评论抽 / 刷礼物送 / 转发抽 / 扣订单号到评论区抽奖
✅ 安全替代：统一说"来参与一下我们的福袋"→打开平台福袋工具→让观众点福袋参与
4主播说："收到货你就去附近小米之家，找店员，让他帮你查验一下是不是正品、有没有问题，全国门店都行的。"这句话的问题是什么？
✅ 正确答案：C — 售后服务不符（虚假宣传）
7月4日最新真实案例！小米官方手表直播间主播说"去附近小米店/小米之家让店员帮你查验一下"，被判定为虚假宣传。实际门店不提供鉴真/验货服务。已申诉但申诉失败。再犯将升级至冻结佣金5%×30天。 📎 案例：违规单号 7658380441224773924（申诉失败）
🚫 高危话术：去门店验货 / 专柜鉴真 / 门店查验 / 线下专柜鉴定 / 小米之家免费检测 / 实体店帮你看
✅ 安全替代：引导查看商品详情页售后说明 / "可以通过官方客服渠道咨询" / 不要替门店做任何承诺
5主播说："我们家的手环比华为好用一百倍，华为手环测心率根本不准，电池也不行，买了绝对后悔。"这属于？
✅ 正确答案：B — 不当竞争+夸大宣传
小米官旗手表直播间6月25日真实案例：因"贬低第三方同类产品"被判定不当竞争。"好用一百倍"属于无依据的夸大描述。两种违规叠加，处罚更重。 📎 案例：违规单号 765545910472747432
🚫 高危话术：XX品牌不行 / XX不如我们 / 买XX肯定后悔 / XX早就落后了 / 比XX好一百倍 / XX测不准
✅ 安全替代：只说自家产品参数 / "我们的屏幕是AMOLED，续航14天，您可以对比一下" / 让消费者自己判断
6主播说："今天下单的，我再额外送你一个耳机包！只有今天有！"但商品详情页完全没有提到耳机包赠品。这属于？
✅ 正确答案：A — 赠品活动信息与宣传不符
这是5-7月最高频的违规类型之一！耳机直播间5月2日真实案例：口播"赠送耳机包"，详情页无此信息，被判违规并处罚：下架商品+冻结佣金5%+中断直播。 📎 案例：违规单号 76352147209546632040 及多个类似案例
🚫 高危话术：买就送XX（详情页没有） / 额外赠送XX / 加赠XX / 只有今天送 / 下单备注送XX
✅ 安全替代：赠品必须先上详情页→确认上线→直播时再口播 / 说"具体赠品信息请看商品详情页"
7主播说："我们家有运费险，退换货你一分钱不用出，来回运费我们都包了。"这句话的问题是什么？
✅ 正确答案：C — 过度承诺，运费险不等于全包运费
运费险通常只赔付首重/基础运费，超额部分或特殊情况下消费者仍需承担费用。说"一分钱不用出""来回都包了"属于过度承诺，可能触发综合服务不符违规。
🚫 高危话术：来回运费都包 / 退换货一分钱不用出 / 送货上门（包邮≠送货上门） / 送货到家
✅ 安全替代：正常说"包邮""有运费险"即可 / "具体运费险保障范围请看详情页"
8主播介绍产品时说："这个手环有降温功能，夏天戴着特别凉快，还能省电，续航比官方标注的还长。"这属于什么违规？
✅ 正确答案：D — 夸大宣传（虚假功效描述）
"降温""省电""节能"属于功能性描述。如果产品实际上不具备这些功能，或没有检测报告佐证，则属于虚假宣传。平台对此类话术严查且基本申诉不回来。
🚫 高危话术：降温 / 省电 / 节能 / 杀菌 / 抗菌 / 除螨（无检测报告时） / 续航比官方长 / 终身质保
✅ 安全替代：严格按产品说明书功能描述 / 有检测报告的功能可以提但要注明"经XX机构检测" / 不确定的功能不说
9主播展示手表时说："看这个表盘，颜色多正！这个金属边框的质感..."但直播画面展示的是银色款，购物车链接卖的是黑色款，且两款外观有差异。这属于？
✅ 正确答案：B — 款式/颜色/图案/形状不符
6月23日小米官方手表直播间真实案例：宣传款式与详情页展示不一致，被判违规。处罚：下架商品+冻结佣金5%。 📎 案例：违规单号 7654339375326398245
🚫 前台注意：展示的产品实物颜色/款式必须与链接商品完全一致 / 不能拿A款展示卖B款 / 美颜滤镜不能改变产品颜色
✅ 安全做法：卖哪个款就展示哪个款 / 切换产品时同步切换链接 / 多个颜色分别展示并说明
10以下哪句话术是完全合规的？
✅ 正确答案：A — 完全合规的描述方式
A选项的特征：(1)描述的是产品真实参数；(2)用"参数在详情页都有标注"引导消费者查看官方信息；(3)没有过度承诺、没有夸大、没有贬低他人。这是最安全的话术风格。
✅ 话术金线原则：说详情页有写的 → 说产品真实参数 → 不确定的不说 → 引导看详情页 → 不替品牌/门店做任何承诺
二 口播红线判断 · 判断题（每题2分，共20分）
👉 判断以下说法或做法是否正确。这些都是主播在镜头前最容易踩的坑。
11直播中口播"终身质保""只换不修""终生售后"，只要主播觉得产品质量好、不太可能坏，说说也无妨。
✅ 正确
❌ 错误
✅ 正确答案：错误
"终身质保""只换不修""终生售后"是平台严查的高危词！除非商品详情页明确标注且品牌方确实提供此服务，否则说出口即违规。这类话术基本申诉不回来。
🚫 这些词在未确认的情况下，一个字都不要说：终身质保 / 终生售后 / 只换不修 / 假一赔三 / 假一赔十
12在管理员评论区发"下单备注送XX赠品"是可以的，只要主播口播没有说违规词就行。
✅ 正确
❌ 错误
✅ 正确答案：错误
管理员评论同样是平台审核范围！小米官方手表直播间曾因管理员在评论区发布的赠品信息与详情页不一致，被判"商品活动信息不规范"。口播、画面、评论三个渠道都会被审核。 📎 案例：违规单号 7648496881592516883
🚫 别以为"口播没说就没事"——画面文字、管理员评论、弹幕回复、商品标题都会被AI扫描
13平台有AI+人工的直播实时审核系统（直播预审），会在直播过程中自动扫描画面和语音，主动推送违规预警。
✅ 正确
❌ 错误
✅ 正确答案：正确
多个违规记录显示违规来源为"直播预审发现"。抖音平台的AI审核系统支持实时画面扫描+语音识别+关键词匹配。千万不要抱有"没人发现"的侥幸心理，你在镜头前说的每一句话都可能被机器识别。
14运营临时通知"今天加个赠品，你口播一下就行，详情页我播完再补"。作为主播，应该听从运营安排先口播。
✅ 正确
❌ 错误
✅ 正确答案：错误
这是赠品类违规的最高频原因！"先说了再补详情页"已经导致多次处罚。作为前台主播，你有权也有责任拒绝此类要求。正确做法：要求运营先更新详情页，确认上线后再口播。口播和详情页的信息差就是违规的直接证据。
✅ 主播自我保护：赠品→先看详情页有没有→没有就拒绝口播→等运营更新后再宣传 / 这是你的合规底线
15"包邮"和"送货上门"是两个不同的概念，说"包邮送到家"或"送货上门"可能触发违规。
✅ 正确
❌ 错误
✅ 正确答案：正确
包邮 = 卖家承担快递费；送货上门 = 快递员送进家门（属于物流增值服务）。两者不是一回事。说"送货上门""送到家门口"可能被判定为过度承诺，基本申诉不回来。
🚫 避免的词：送货上门 / 送到家 / 送货到家
✅ 安全表达：包邮 / 快递包邮 / 顺丰包邮
16使用美颜滤镜让产品看起来更精致是每个直播间都在做的，平台不会因此判定违规。
✅ 正确
❌ 错误
✅ 正确答案：错误
违规记录中明确提到"通过美颜、不科学的实验等方式夸大商品实际效果"属于违规。适度的直播间美颜用于主播本人是可以的，但如果滤镜改变了产品颜色、质感、外观与实物产生显著差异，就会触发"款式/颜色不符"或"夸大宣传"违规。
17第一次违规预警虽然叫"预警"，但也可能伴随"冻结佣金5%×30天"的实际经济损失。
✅ 正确
❌ 错误
✅ 正确答案：正确
预警级别并非"没事"。从多个案例看，预警处罚已经包含：下架商品+冻结佣金5%+中断直播。再犯将升级至冻结佣金15%+关闭商品分享。处罚升级路径：警告 → 冻结5% → 冻结15%+下架 → 永久关闭分享 → 保证金额提升至3000元。
18介绍产品时，为了增强信任感，可以说"我自己也在用""我家人都在用"，这是增加亲和力的手段，不违规。
✅ 正确
❌ 错误
✅ 正确答案：错误
如果主播确实在使用，分享个人真实体验一般问题不大。但如果以此来佐证产品功效（如"我用了感觉XX病都好了""我戴了睡眠质量提升80%"），且表述夸张无依据，就可能构成虚假宣传。分享真实感受可以，但不能替代产品客观参数和功能描述。
✅ 安全做法：区分"个人感受"和"产品功能" / "我个人觉得挺好用" vs "这个能治XX"是两回事
19收到平台的"直播预审"违规预警推送后，正确的做法是：立即查看预警内容→停止相关话术→通知运营→继续正常直播→播后复盘整改。
✅ 正确
❌ 错误
✅ 正确答案：正确
直播预审预警是平台给的"改正机会"，不是最终处罚。标准流程：立即查看→停止违规话术→通知运营→继续直播（换安全话术）→播后复盘。不处理继续播可能导致处罚升级。也不需要在直播中跟观众抱怨"平台审核太严"。

26【口播应急】你正在直播小米手环10Pro，突然有观众刷屏提问："主播主播，这个手环能测血压吗？我妈妈有高血压。"实际上该手环没有血压监测功能。你应该怎么回答？
✅ 正确答案：C — 如实告知，不虚构功能
虚构产品功能属于虚假宣传/夸大宣传，手表直播间已有"五十米防水"的教训。A选项会直接触发违规，且消费者收货后发现没有血压功能会投诉，可能导致"劣质/低质"判罚。C的做法既诚实又合规，同时给消费者提供了替代建议。
✅ 话术模板：如实告知有没有 → 说明实际有的功能 → 引导到其他合适产品 → 不确定的功能绝对不说有
27【赠品风险】运营在你直播中途跑过来小声说："今天业绩不好，你口播一下，就说现在下单额外送一条硅胶腕带，快！"你扫了一眼详情页，发现没有这条赠品信息。你应该？
✅ 正确答案：B — 拒绝，要求先更新详情页
这是赠品类违规的最高频场景！耳机、手表、手环直播间已经因此被罚多次。C选项（模糊承诺）和D选项（评论区代替）也都有违规案例——评论区发布同样会被审核。正确做法：详情页更新只需1-2分钟，更新完再口播，合规且不影响业绩。 📎 案例：多个赠品类违规的penalty包含"冻结佣金5%+中断直播"
28【售后雷区】观众问："万一手表坏了怎么办？去哪修？你们管不管？"以下四种回答中，哪个最安全合规？
✅ 正确答案：D
A包含"全国联保""线上线下都管"（超过详情页承诺）→ 可能触发售后服务不符；B包含"只换不修"（在详情页未标注的情况下高危）；C就是7月4日被罚的那句话（门店查验服务不存在）→ 申诉都失败了。D安全的原因：引导消费者去查看官方信息，不替品牌方/门店做任何承诺。 📎 案例：违规单号 7658380441224773924（申诉失败）、7642750111487148331、7543509078755770086
✅ 售后问题万能安全回答："具体售后政策可以在详情页查看哦" / "包装里有售后卡，上面有详细的保修说明"
29【抽奖翻车】你一时兴起，在直播中说了句："兄弟们，今天开心！评论区的宝宝们，扣一波'想要'，我抽5个送手环！"话刚说完你意识到这可能违规。接下来你应该怎么做？
✅ 正确答案：C — 立即止损，切换到福袋工具
说错话不可怕，关键是要立即纠正。C的做法：承认口误→切换到合规方式（福袋）→继续正常直播。平台的直播预审系统关注的是持续违规行为，及时纠正可能只收预警而非处罚。A和D的做法是持续违规，会加重处罚。B过于极端，不需要下播。
✅ 说错话后的正确操作：承认口误→立即切换到合规话术→通知运营记录→播后复盘→下次注意
30【竞品陷阱】观众在弹幕刷屏："主播，华为手环9和你们这个小米手环10Pro哪个好？说实话！"以下哪种回应方式最合规且最专业？
✅ 正确答案：B — 客观陈述自身优势，让消费者自己判断
这是最专业也最安全的回答方式。特征：(1)只列举自家可验证的参数；(2)用"您可以对比一下"而非"我们比XX好"；(3)结尾"适合自己的才是最好的"体现了对消费者选择的尊重。A和C都属于不当竞争（贬低竞品），6月已有真实处罚案例。 📎 案例：小米官旗手表直播间不当竞争违规（765545910472747432）
