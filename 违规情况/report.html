以下违规类型近期频发，基本申诉不回来，请务必传达至所有直播间运营及主播

🔥
降温 / 省电 / 节能 / 终身质保
降温省电节能终身质保
老生常谈的问题，但仍有商家在详情页呈现、主播口述中使用此类话术。平台严查且申诉不回来。
📢
假一赔三 / 质保X年 / 只换不修 / 全国联保
假一赔三质保X年只换不修全国联保
此类承诺类话术，详情页需要明显标注出来，且与口播或直播页面展示完全一致。6月在手表类直播间持续高发，申诉不回来。
📦
送货上门 / 运费险
送货上门运费险送货到家
包邮 ≠ 送货上门/送货到家，运费险 ≠ 来回不需要出运费。正常说"包邮"和"有运费险"即可，不要过度承诺。此类基本也申诉不回来。
🆕
诱导互动（6月新出现）
诱导互动引导评论利益诱导
6月小米官旗手表直播间出现2次诱导互动违规。以抽奖、赠送、利益回馈等方式诱导用户评论/点赞/关注的行为被平台判定违规。此类基本申诉不回来。抽奖/赠品活动必须使用平台福袋工具。
🔧 行动项：所有直播间禁止使用"扣1抽奖""评论抽""点赞送"等诱导话术，统一使用平台福袋工具。
🆕
不当竞争（6月新出现）
贬低竞品不当比较负面评价
6月25日小米官旗手表直播间因不当竞争被预警。直播中贬低第三方同类产品，或对某同类产品表达负面评价被判定违规。避免在直播中对比友商产品时使用贬低性语言。
🔧 行动项：话术培训：对比竞品时只说自身优势，不做竞品负面评价。
🆕

🔧 行动项：所有直播间禁止使用"去门店验货""线下鉴真""专柜查验"等表述。除非确认门店确实提供该服务，否则不得承诺。

关键发现（6月更新）

  • 累计违规32次，涉及6个直播间，有效违规29次（扣除3次已撤销）
  • 小米官方耳机直播间累计最多（10次/32%），仍是重点整改对象
  • 售后服务不符仍是最突出问题（14次/45%），承诺"全国联保""一年质保"等话术反复触发
  • 赠品活动信息与宣传不符次之（10次/32%），赠品未在商品详情页体现
  • 🆕 新增直播间小米官旗手表直播间（6次违规），集中在诱导互动和售后服务不符
  • 🆕 出现4种新违规类型：诱导互动、违规买赠、款式/颜色不符、不当竞争
  • 手表类直播间（官方手表+官旗手表）合计11次，成为6月违规重灾区
  • 🆕 6月新增违规10次，日均0.33次，整体频率较5月有所下降
  • 新增违规均为"预警"级别，暂无严重处罚（冻结佣金/关闭分享），说明整改初见成效
  • 🆕 7月4日小米官方手表直播间新增1次"售后服务不符"违规：主播口播引导消费者"去附近小米店查验"，被判定为虚假宣传。已申诉但申诉失败
  • 处罚升级路径：警告 → 冻结佣金5% → 冻结佣金15% → 关闭商品分享 → 提升保证金至3000元

整改建议（6月更新）

  1. 规范话术：不得在未确权情况下承诺"全国联保""终生售后""一年质保""无忧质保"等售后表述
  2. 赠品信息同步：所有赠品需在商品详情页明确展示，直播间口播与详情页严格一致
  3. 建立审核机制：每场直播前对文案/口播脚本进行合规审核
  4. 🆕 官旗手表直播间专项整改：新直播间需立即建立合规话术体系，重点解决诱导互动和售后服务不符问题
  5. 🆕 禁止诱导互动话术：严禁"扣1抽奖""评论抽""点赞送"等诱导性话术，抽奖统一使用平台福袋工具
  6. 🆕 竞品对比规范：直播中对比友商只说自身优势，不做竞品负面评价，避免触发不当竞争违规
  7. 规范抽奖流程：直播间任何抽奖/赠品活动必须使用平台官方福袋工具
  8. 耳机直播间持续关注：该直播间已面临保证金+佣金双重压力，需持续监控
  9. 申诉跟进：部分违规有14天申诉期，逾期未申诉视为放弃，需安排专人跟踪
  10. 🆕 定期合规培训：建议每月1次违规案例分析会，将最新违规类型及时传达给所有运营和主播
  11. 🆕 7月4日新增：禁止"去门店验货/鉴真"类话术 — 不要说"去附近小米店/小米之家让店员帮你查验"，实际门店并不提供此项服务，被认定为虚假宣传。此类话术在所有直播间全面禁用
